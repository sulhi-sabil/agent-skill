# Supplementary Repositories

This directory contains references to supplementary repositories used by the CMZ agent.

## Repositories

- [oh-my-opencode](https://github.com/code-yeongyu/oh-my-opencode.git) - Agent harness with Sisyphus orchestration
- [superpowers](https://github.com/obra/superpowers.git) - Systematic debugging framework
- [UltraRAG](https://github.com/OpenBMB/UltraRAG.git) - MCP-based RAG framework
- [system_prompts_leaks](https://github.com/asgeirtj/system_prompts_leaks.git) - Prompt optimization reference

## Usage

Clone these repositories as needed:
```bash
cd supplementary-repos
git clone https://github.com/code-yeongyu/oh-my-opencode.git
git clone https://github.com/obra/superpowers.git
git clone https://github.com/OpenBMB/UltraRAG.git
git clone https://github.com/asgeirtj/system_prompts_leaks.git
```
